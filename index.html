6SEG SCRIPT

6本のセグメントからなる、存在しなかった英数字表示規格。

SCRIPT MODE

Text to 6SEG

暗号化しない通常モードです。小文字は大文字に変換されます。対応していない文字は ? に置換されます。

OUTPUT

Visible Glyph PNG

PNGの中に隠しデータは入れていません。描かれた6セグ文字そのものを読み取ります。X投稿には「X用PNG ZIP保存（読取対応）」を使ってください。長文は小さめのPNGに自動分割され、ZIPにまとめて保存されます。

READ SCRIPT

PNG to Text

この読み取りは、このアプリで保存した白黒PNGを想定しています。複数枚のX用PNGもまとめて読み込めます。v0.3.1以降は縮小画像にも前より強くしていますが、色変更や強い圧縮では失敗する場合があります。

LOCK MODE

Strong encrypted 6SEG PNG

AES-GCM 256bit + PBKDF2-SHA-256。PNGには隠しメタデータではなく、暗号データを6SEG文字として描画します。

LOCK OUTPUT

Encrypted Glyph PNG

この画像は読もうとしても元文は読めません。復号には同じパスワードが必要です。

UNLOCK MODE

LOCK PNG to Text
